Introducing a New Way
to Find Textbooks on Campus.


BookCycle is commited to making education more accesible to everyone.

We are a platform that allows George Washington students to exchange textbooks within the safety of their campus. Students can donate and swap textbooks on campus to reduce the number of financial insecurities in higher education and academia. BookCycle not only aims to reduce financial strain, but also reduce the environmental impact of purchasing new textbooks every semester.

Building our Project


To build BookCycle, we made a number of choices for our UX/UI Design and Front-end to make it as user-friendly as possible. We used React which is the Front-end JavaScript framework. Axios which is a JavaScript library to make HTTP requests to the API and is used for the API requests to endpoints in the backend, and Material UI for user components.

For the Backend/API we used Node.js as the JavaScript runtime environment, Express as the framework to build REST API, PostgreSQL as the Relational Database, TypeORM to Connect Node and Postgre, BeautifulSoup to web scrape GW courses.


To read more about our Front-end, Backend, and other documentation go to our writings.
